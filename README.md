MoodSphere – Mood Based Room Experience

Project Overview
This is my third internship project for the DecodLabs Web Development Internship.
MoodSphere is an interactive web project based on **DOM manipulation using JavaScript**. The website changes the room experience based on the selected mood.

Technologies Used
* HTML5
* CSS3
* JavaScript (DOM Manipulation)

Objective
The objective of this project is to practice JavaScript DOM manipulation by creating dynamic changes in the UI based on user interaction and mood selection.

Features
* Dynamic room theme changes based on mood
* Interactive user interface
* DOM manipulation for real-time updates
* Simple and responsive design

How to Run the Project
Open the `index.html` file in any modern web browser to use the application.

Note
This is a beginner-level interactive project focused on understanding JavaScript DOM manipulation and dynamic UI behavior.
